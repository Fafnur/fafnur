Stack

Мой стек технологий, который я использую и популизирую:

  • Ng - CLI для создания приложений и компонентов angular
  • Nx - монорепозиторий для разработки enterprise приложений
  • NGRX - реализация Redux в Angular
  • Universal - Server-side rendering в Angular
  • Apollo Angular - GraphQL клиент для Angular
  • Nativescript - разработка нативных мобильный приложений на Angular
  • NestJS - Node.js платформа для создания эффективных, надежных и масштабируемых серверных приложений.
